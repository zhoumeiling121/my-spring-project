外卖后台管理系统 - 后端

技术栈
- Java 8
- Spring Boot 2.7.0
- MyBatis Plus
- MySQL 8.0

主要功能
- 员工登录、登出、管理
- 菜品分类管理
- 菜品信息管理

如何启动
1. 创建 MySQL 数据库 reggie
2. 修改 application.yml 中的数据库密码
3. 启动 ReggieApplication.java
4. 访问 http://localhost:8080/backend/page/login/login.html

我的工作
- 独立完成后端接口开发
- 使用 MD5 对用户密码加密
- 使用 LambdaQueryWrapper 简化数据库查询
